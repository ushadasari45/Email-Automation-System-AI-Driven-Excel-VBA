# Email-Automation-System-AI-Driven-Excel-VBA

This project demonstrates an automated workflow for sending high-volume, personalized emails directly from Microsoft Excel using VBA (Visual Basic for Applications) and Outlook. By leveraging AI to generate precise automation logic, this system eliminates manual data entry, reduces human error, and significantly increases operational efficiency.

🚀 Features

Automated Personalization: Dynamically pulls recipient names and data from Excel cells to craft custom email bodies.

Dynamic File Attachment: Automatically maps and attaches specific PDF documents to their respective recipients based on folder paths.

Batch Processing: Capable of sending hundreds of emails in seconds with a single click.

AI-Optimized Code: Developed using advanced prompt engineering to create robust, error-free VBA macros.

Error Handling: Basic validation to ensure file paths exist before attempting to send.

🛠️ Technical Tools Used
Language: VBA (Visual Basic for Applications)

Host: Microsoft Excel

Integration: Microsoft Outlook API

Development Assistant: AI (Prompt Engineering for Scripting)

📂 How It Works
Data Preparation: The Excel sheet contains columns for Recipient Name, Email Address, and Attachment Path.

Path Mapping: A macro scans a local folder and automatically populates the Excel sheet with the correct file paths for each PDF.

The Mail Loop: The main VBA script loops through the rows, initializes an Outlook mail item, attaches the unique file, and triggers the .Send command.

📊 Business Impact
Time Savings: Reduced a 3-hour manual task to a 30-second automated process.

Accuracy: Achieved 100% accuracy in matching attachments to recipients, eliminating the risk of data privacy leaks.

Scalability: The system can be scaled from 10 to 1,000+ emails without any additional manual effort.
